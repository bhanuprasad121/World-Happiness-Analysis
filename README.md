# World Happiness Report 2019 Analysis

This project analyzes the World Happiness Report 2019 dataset using Python libraries such as Pandas, Matplotlib, and Seaborn in Google Colab.

## Project Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- 6 Data Visualizations
- Correlation Analysis
- Insights Report

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset
World Happiness Report 2019 Dataset
